"알아?" / "네?" / "당신 '배달꾼'이야." / "네?" / "배달한다고." / "뭘요?" / "그걸 우리가 어떻게 알아. 그건 당신 일이야." / "전 그냥 블로거인데요." / "'배달꾼'이라고, 썅. 오늘 일이 여기 있잖아." / "무슨 일이요?" / "이 신발. 그냥 둘 거야?" / "네?" / "가방에 넣어." / "네?" / "며칠 전에 가방 샀잖아. 페북에 새 초딩가방 샀다고 자랑도 했잖아. 몰라?" / "아네, 가방 샀는데요." / "이 신발 넣으려고 샀잖아." / "아닌데요?" / "아니긴 뭐가 아니야. 이 신발 넣으려고 산 거야. 당신이 샀잖아, 그래서. 그렇게 결정되어 있었어. 몰라?"

크로스백에 책을 넣고 산책을 다니니 몸이 계속 비뚤어지는 것 같았습니다. 며칠 전에 이마트에서 필라 초딩가방을 하나 샀습니다. 그 가방을 말하는 것 같습니다.

"어서 넣어." / "네?" / "이 신발 말이야. 어서 넣어. 시간이 별로 없어." / "..." / "넣으라니까. 썅."

갈매기의 눈초리가 무서웠습니다. 구두를 집어서 유리와 흙을 털고 가방에 넣었습니다. 가방을 다시 지고 잠자코 있었습니다.
